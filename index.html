Flex, the layout you'll want to use next

A powerful layout mechanism that lets you solve common responsive web design problems with ease.

Learn how flexbox rolls

Take a look at the Udemy course and the git respository

Flexbox is a new layout module in CSS3 that lets you lay out, align and distribute space among items within a container, even when their sizes are unknown or dynamic.

In this course you'll learn to:

  • Build flexible designs with less code
  • Align items both vertically and horizontally with no pain
  • Order items as your will without having to change your HTML
  • Build responsive layouts quite easily

All the content of this course will be available in this page and which you can access on github.
